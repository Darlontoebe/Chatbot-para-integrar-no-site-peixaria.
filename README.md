# 🐟 Z&Z Peixaria - Chatbot de Atendimento

Um chatbot simples, bonito e funcional de atendimento ao cliente para a **Z&Z Peixaria**, desenvolvido apenas com HTML, CSS e JavaScript puro.

### ✨ Funcionalidades

- Interface moderna estilo WhatsApp
- Botão flutuante elegante
- Respostas automáticas sobre:
  - Preços dos produtos (Filé de Tilápia, Empanado, Lambari, Traíra Inteira e Bolinho de Bacalhau)
  - Endereço da peixaria
  - Contato via WhatsApp
  - Horários de atendimento
- Totalmente responsivo (funciona bem no celular)
- Fácil de usar e entender

### 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
